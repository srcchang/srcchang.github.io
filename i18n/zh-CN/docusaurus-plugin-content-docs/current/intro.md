# 关于 Project SRC

* 一个手机应用程式开发者制作的 Side Project
* 针对自己的需求制作了一系列的 Mobile Apps
* Firebase serverless 架构
* Flutter 跨平台框架

## 游乐园类

### 日本环球影城 USJ 排队攻略

[![usj](./img/usj_logo.png)][1]

提供日本环球影城 Universal Studios Japan (USJ) 的即时排队等待时间与入园人数预估，方便您规划每一次的游玩。

### 东京迪士尼海洋 TDS 排队攻略

[![tds](./img/tds_logo.png)][2]
